### 2. Laboratorio 1: Fundamentos de Python para Usuarios de R

**Principales Diferencias Sintácticas entre R y Python**
- **Indexación Base Cero**: Python comienza la indexación en 0.
- **Indentación**: Importancia de la indentación en Python en comparación con el uso de llaves en R.
- **Mutabilidad e Inmutabilidad**: Diferencias en el manejo de estructuras mutables e inmutables.

**Tipos de Datos y Estructuras en Python**
- **Estructuras en Python**: Listas, tuplas, diccionarios y conjuntos.
- **Comparación con Estructuras en R**: Vectores, listas, matrices y data frames.

**Estructuras de Control**
- Condicionales: `if`, `elif`, `else`.
- Bucles: `for` y `while`.
